What is the sample size calculator?
The sample size calculator is a tool that determines the sample size for infinite population.
How to calculate sample size?
To calculate sample size you should use the following formula:
sample size = ((z value)^2 * percentage of population * (1 - percentage of population)) / (confidence level)^2
